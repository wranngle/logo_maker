# Contributing to logo_maker

First off, thank you for considering contributing! We appreciate your time and effort.

## Getting Started

1. Fork the repo and clone it locally.
2. Run the project install command to set up your local environment.
3. Create a branch for your feature or bug fix.

## Testing & Style

- **Tests:** Please run the test suite to ensure everything works as expected before submitting your changes.
- **Code Style:** We aim for consistency. Try to match the existing code style and run any provided linters.

## Submitting a PR

- Keep your Pull Request focused on a specific feature or bug.
- Include a clear description of what changed and why.
- Link any relevant issues.

## Questions vs. Issues

- **Found a bug or want to request a feature?** Open a GitHub Issue.
- **Have a question or want to discuss an idea?** Join us in GitHub Discussions.

Thanks for helping make logo_maker better!
